# LOCUTRA

A geography quiz game for learning Chinese administrative divisions.

## Quick Start

Requirements:
- Node.js >= 18
- Modern browser

Quick start with pre-compiled WebAssembly:
```bash
SKIP_RUST=true npm install
npm run build:web
npm run dev
```

For WebAssembly development:
```bash
npm install
npm run build
```
